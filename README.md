# Emoji Icons

A fast, searchable emoji browser. Click any emoji to copy it to your clipboard.

Built with vanilla JS + Vite, using [`unicode-emoji-json`](https://github.com/muan/unicode-emoji-json) for emoji data.

## Responsive Breakpoints

| Breakpoint | Range | Grid cell | Emoji size | Header |
|---|---|---|---|---|
| Mobile | `< 640px` | 56px | 1.75rem | 2 rows (title+toggle / search) |
| Tablet | `640px – 1024px` | 64px | 2.25rem | 1 row |
| Desktop | `> 1024px` | 72px | 2.625rem | 1 row |

Hover scale is disabled on touch devices.

## Usage

```sh
pnpm dev      # dev server
pnpm build    # production build → dist/
pnpm deploy   # build + deploy to GitHub Pages
```
